embedodon

embedodon is a simple client-side typescript library to render a Mastodon user’s public timeline.

  • source: github.com/danieldickison/embedodon

example

A simple example for showing the author’s toots, using a standard built-in color scheme: 

  <script type="module">
    import { EmbedodonElement } from 'https://unpkg.com/embedodon@1.0.0-beta.0/dist/index.js'
    customElements.define('embedodon-example', EmbedodonElement)
  </script>
  <embedodon-example username="@dand@mastodonmusic.social" class="standard"></embedodon-example>

You can also customize the appearance using CSS with custom properties, ::part() selectors, and adoptedStyleSheets.
