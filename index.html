embedodon

embedodon is a simple client-side typescript library to render a Mastodon user’s public timeline.

  • source: github.com/danieldickison/embedodon

example

A simple example for showing the author’s toots: 

  <script type="module">
    import { EmbedodonElement } from 'https://unpkg.com/embedodon@1.0.0-beta.0/dist/index.js'
    customElements.define('embedodon-example', EmbedodonElement)
  </script>
  <embedodon-example username="@dand@mastodonmusic.social"></embedodon-example>

You can also add custom CSS rules. See the index.html source for how this is accomplished.
